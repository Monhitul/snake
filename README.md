这是大一暑假为了进学校的一个技术小组写的一个小玩意，贪吃蛇小游戏，后来也算是如愿进了，虽然后来转软件开发了。

实现暂停，加速，障碍，计时，计分，闯关，还有我最喜欢的第三关的创意，当时真的是灵光一现就想到这一关。

第一关就是普普通通的地图，一个方框圈起来，撞到墙或者吃到自己就输，吃到10个食物可过关。第二关是在第一关基础上增加了障碍物，是一个笑脸，吃到10个可过关。第三关是直接第一关的那个方框，然后玩家自己画障碍物，就是你所走过的地方都会成为障碍物，而同样，只要吃到10个食物就可过关。

51单片机，C语言，LCD12864。

包含源代码，电路图，效果图，说明文档。
